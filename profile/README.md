# 🥊 About Sparring

*Sparring은 사용자 건강 데이터를 기반으로 식후 혈당 변화를 예측하고,  
맞춤형 건강 피드백을 제공하는 AI 기반 헬스케어 플랫폼입니다.*

<br/>

## ✨ Main Features

- 🩸 식후 혈당 변화 예측
- 📊 건강 데이터 시각화
- 🤖 AI 기반 건강 피드백
- 🏃 지속 가능한 건강 습관 관리
